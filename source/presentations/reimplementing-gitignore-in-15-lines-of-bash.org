* What is a .gitignore file                                           :slide:

** A simple shell style syntax (globbing) for specifying files

** Some examples:

#+begin_src
# Specific filenames
.emacs.desktop
.emacs.desktop.lock

# Directories
auto/

# Wildcard Patterns
*.gch
*.beam
*.com
*.class

... some others
#+end_src

To see more check out https://github.com/github/gitignore
